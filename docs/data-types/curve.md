## Overview

`Curve` represents one or more procedural splines defined by points and edges.

---

## Contains

- Control points
- Edges defining spline paths
- Multiple independent splines

---

## Usage

- Curve generator nodes output `Curve`
- Modifier nodes transform or resample curves
- Conversion nodes can transform curves into meshes

---

## Notes

- Curves may contain multiple disconnected paths
- Many curve operations operate per-spline
