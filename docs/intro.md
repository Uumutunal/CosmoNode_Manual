# CosmoNode

Welcome to the **CosmoNode documentation**.

CosmoNode is a node-based procedural modeling toolkit built directly into Unity, designed for fast iteration, reusable workflows, and procedural content creation.

These docs explain how CosmoNode works, from core concepts and data types to individual node behavior and examples. Whether you're an artist or a developer, you’ll find everything you need to start building procedural geometry inside Unity.

Get started by learning the basics, or jump straight into the node reference.
